### 3. 확인

```sql
SELECT
  hebrew,
  meaning,
  letters,
  CASE WHEN icon_svg IS NOT NULL THEN 'EXISTS' ELSE 'NULL' END as icon_svg_status,
  emoji,
  array_length(related_words, 1) as related_words_count
FROM words
WHERE verse_id = 'genesis_1_1'
ORDER BY position;
```
