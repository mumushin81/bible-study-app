# 어근 발음 표시 변경 노트

## 주요 변경 내용
- 히브리어 발음을 영어 음절로 변환
- 모음 기반 음절 생성 로직 개선
- 직관적인 영어 발음 표현

## 변환 규칙
- 모음 변환 맵:
  - 'a' → 'ah'
  - 'e' → 'eh'
  - 'i' → 'ee'
  - 'o' → 'oh'
  - 'u' → 'oo'

## 적용 위치
- 어근 분석 섹션
- 히브리어 글자별 발음 표시

## 목적
- 사용자의 히브리어 발음 이해 용이성 증대
- 직관적인 영어 음절 표현
- 학습자 친화적 발음 가이드

## 예시
- 원래 발음: `ta`
- 변환 발음: `tah`

- 원래 발음: `ki`
- 변환 발음: `kee`

## 향후 개선 방향
- 더 정교한 발음 변환 알고리즘 개발
- 국제 음성 기호(IPA) 지원 검토
