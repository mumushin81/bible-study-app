# Replicate MCP 이미지 생성 유틸리티 업데이트

## 주요 변경 사항
- Replicate MCP 가이드라인 엄격 준수
- 단어 중요도 판별 로직 구현
- 두 가지 모델 지원:
  1. `lucataco/animate-diff` (중요 단어, GIF)
  2. `black-forest-labs/flux-schnell` (비중요 단어, JPG)

## 핵심 기능
- 단어 중요도 자동 판별
- 동적 프롬프트 생성
- 모델 선택 및 파라미터 설정
- Supabase Storage 자동 업로드
- 파일명 해시 생성

## 단어 중요도 판별 기준
### 중요 단어 (GIF 생성)
- 명사, 동사 문법
- 핵심 신학 용어
- 구체적 명사

### 비중요 단어 (JPG 생성)
- 전치사
- 접속사
- 관사
- 문법 표지

## 프롬프트 생성 규칙
- 단어 의미 시각화
- 파스텔 색상 강조
- 16:9 비율
- 하단 20% 공백
- 성경적 예술 스타일

## 주의사항
- 어두운 색상 사용 금지
- 다채로운 색상 필수
- JPG 형식 고수
- 고품질 이미지 생성

## 향후 개선 방향
- 프롬프트 생성 알고리즘 고도화
- 이미지 생성 실패 시 대체 메커니즘 추가
- 더 많은 단어 분류 패턴 추가
